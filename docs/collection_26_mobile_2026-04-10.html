Apple Car Key Support Coming to Lexus Vehicles
Apple Car Key 支援即將來到 Lexus 車型
Evidence suggests Apple is preparing to bring Car Key support to Lexus vehicles, MacRumors has discovered. Code references to Lexus were found in Apple's backend code, indicating the Toyota-owned luxury brand is being added to Apple's Car Key backend. The discovery mirrors how Toyota's own Car Key support was first uncovered, before the feature went live for the 2026 RAV4 in February. It is unclear when the feature will roll out to customers. Lexus already offers its own app-based Digital Key system, but unlike Apple Car Key, it requires an active connection to Toyota's servers to function and has not historically worked via Apple Wallet. Apple Car Key stores a digital key directly in the Wallet app and uses NFC for unlocking, with an Express Mode that allows access without authentication.
MacRumors 發現證據表明 Apple 準備為 Lexus 車型引入 Car Key 支援，代碼中發現的 Lexus 參考資料表明這個豐田旗下品牌正被新增至 Apple Car Key 後端。
