# Chip Specifications Upload

## 如何上傳晶片規格 / How to Upload Chip Specs

請將 Qualcomm 和 MediaTek 旗艦晶片的規格文件上傳到此目錄。
Please upload Qualcomm and MediaTek flagship chip specification documents to this directory.

## 支援的格式 / Supported Formats

- PDF 格式的規格表
- Word (.docx) 文件
- Excel (.xlsx) 比較表
- 純文字 (.txt) 規格
- JSON 格式規格

## 命名規則 / Naming Convention

```
{brand}_{chip_model}_{year}.{ext}
```

例如 / Examples:
- `qualcomm_snapdragon_8_elite_2024.pdf`
- `mediatek_dimensity_9400_2024.pdf`
- `chip_comparison_q1_2025.xlsx`

## 當前已載入的規格 / Currently Loaded Specs

*尚無規格文件。請在此目錄上傳晶片規格，agent 將在下次執行時自動讀取並整合到分析中。*
*No spec files yet. Upload chip specs here and the agent will automatically read and integrate them in the next run.*

## 分析維度 / Analysis Dimensions

- CPU 性能與核心配置
- GPU 性能
- NPU/AI 加速引擎性能（TOPS）
- 數據機能力（5G/6G）
- 記憶體頻寬
- 功耗與散熱效率
- 安全功能
- 相機 ISP 能力
- Wi-Fi / Bluetooth 連接
- 製程節點
